## Culqi-node

Wrapper de los web service de <a href="https://culqi.api-docs.io" />Culqi</a>.

Operaciones disponibles (v1.2 de Culqi):
  * Crear token
  * Crear cargo
  * Consultar cargo
  * Crear plan
  * Crear suscripción
  * Cancelar suscripción
  * Devolver cargo
